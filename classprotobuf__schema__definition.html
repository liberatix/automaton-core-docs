◆ add_message()

void protobuf_schema_definition::add_message ( int  message_id)
virtual

Used to add an already created message/enum schema to this schema. The message/enum should first be created and ready (fields/ nested messages/ enums or enum values must be added) before calling this function. Enum can be added to another message and if no message_id is provided in add_enum or message_id = -1, enum will be added globally. If message/enum with the given id doesn't exist, exception will be thrown.

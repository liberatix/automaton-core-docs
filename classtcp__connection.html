◆ connect()

void tcp_connection::connect ( )
virtual

This function is used in the constructor. It is used to start async_connect with the peer. It can be used outside the constructor if the connection didn't happen or was closed or if disconnect() was first used. If a successful connection was made, start_listening() and handler's on_connected() will be called.

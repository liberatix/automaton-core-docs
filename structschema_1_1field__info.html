Detailed Description

Struct representing one field in a message schema (field schema). Does NOT contain value, only describes a field. Tag - unique number within a message schema. Identifies the field. Type - one of the allowed data types. If it is a message or enum type, its message or enum type should be stored in fully_qualified_type. Name - unique name within a message schema. Identifies the field. Fully_qualified_type - specifies the type of message and enum fields. It is ignored for scalar types.
